# Maze Core\cb1 \
\
\
\cb3 **CRT handheld sonar maze for #GitHubGameOff 2025 \'96 Theme: WAVES** \cb1 \
\cb3 Ping **expanding waves** to reveal procedural mazes! Dodge pathfinding red enemies, find yellow EXIT. Every 5 levels: Genre pivot to raycasted 3D FPS **HYPERSPEED** \'96 strafe/turn, collect +LIVES! Endless levels, hard mode (longer cooldowns), local leaderboards, boot seq, shake/invuln/SFX. **Single-file HTML5 \'96 browser/mobile-ready!** v17.\cb1 \
\
\cb3 ## How to Play\cb1 \
\cb3 - **Insert Coin** \uc0\u8594  Tutorial \u8594  Level 1.\cb1 \
\cb3 - **2D Mode**: Move blindly; **PING SONAR** (cooldown bar) reveals green walls, red enemies (avoid touch!), yellow EXIT. Reach it to level up.\cb1 \
\cb3 - **Hyperspeed (Lv5,10...)**: 3D first-person maze \'96 collect floating +LIVES, navigate to EXIT (compass arrows if off-screen). Ping flashes full brightness reveal!\cb1 \
\cb3 - Die 3x? Game Over (high score entry). **Hard Mode**: Slower pings.\cb1 \
\cb3 - **Goal**: Chase endless high score! **Start Btn** toggles overhead Map view (both modes) or Pause (2D).\cb1 \
\
\cb3 ## Controls\cb1 \
\
\cb3 ### Desktop (Keyboard \'96 WASD + Arrows both work!)\cb1 \
\cb3 **2D Top-Down (Most Levels):**\cb1 \
\cb3 | Keys | Action |\cb1 \
\cb3 |----------|--------------|\cb1 \
\cb3 | **W** / **\uc0\u8593 ** | Move Up |\cb1 \
\cb3 | **S** / **\uc0\u8595 ** | Move Down |\cb1 \
\cb3 | **A** / **\uc0\u8592 ** | Move Left |\cb1 \
\cb3 | **D** / **\uc0\u8594 ** | Move Right |\cb1 \
\cb3 | **Space** | Sonar Ping (Reveal Waves) |\cb1 \
\cb3 | **Enter** / **Start Btn** | Toggle Map / Pause |\cb1 \
\
\cb3 **3D Hyperspeed FPS (Every 5th Level):**\cb1 \
\cb3 | Keys | Action |\cb1 \
\cb3 |----------|--------------------|\cb1 \
\cb3 | **W** / **\uc0\u8593 ** | Forward |\cb1 \
\cb3 | **S** / **\uc0\u8595 ** | Backward |\cb1 \
\cb3 | **A** / **D** | Strafe Left/Right |\cb1 \
\cb3 | **\uc0\u8592 ** / **\u8594 ** | Turn Left/Right |\cb1 \
\cb3 | **Space** | Sonar Flash (Full Reveal) |\cb1 \
\cb3 | **Enter** / **Start Btn** | Toggle Map View |\cb1 \
\
\cb3 **Menus / High Score Entry / Settings:**\cb1 \
\cb3 | Keys | Action |\cb1 \
\cb3 |-------------------|---------------------------------|\cb1 \
\cb3 | **\uc0\u8593 ** / **\u8595 ** | Select Up/Down (or Cycle Letters) |\cb1 \
\cb3 | **\uc0\u8592 ** / **\u8594 ** | Move Cursor / Toggle Settings |\cb1 \
\cb3 | **Space** / **Enter** | Confirm / Submit Name |\cb1 \
\
\cb3 ### Mobile / Touch\cb1 \
\cb3 | Control | 2D Mode | 3D Mode |\cb1 \
\cb3 |------------------|----------------------|----------------------------------|\cb1 \
\cb3 | **D-Pad** | 4-Way Move | Forward/Back + Strafe L/R |\cb1 \
\cb3 | **Sonar Btn** | Tap: Ping Reveal | Swipe L/R: Turn (like analog stick) |\cb1 \
\cb3 | **Start Btn** | Toggle Map / Pause | Toggle Map View |\cb1 \

\cb3 **Pro Tip**: Auto-resizes fullscreen. Touch-optimised dpad/sonar. Map shows full maze + your position/facing!\cb1 \
